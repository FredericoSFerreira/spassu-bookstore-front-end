## Iniciando o projeto

- Clonar repositório
```shell
git clone https://github.com/FredericoSFerreira/spassu-bookstore-front-end
````

- Após clonar o repositorio, basta rodar o comando abaixo, para instalar as dependências:
```shell
npm install
````

- Subir o projeto:
```shell
npm run serve
````

### Acessar o Projeto
```bash
http://localhost:8080
````
